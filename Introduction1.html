Web development broadly refers to the tasks associated with developing websites for hosting via intranet or internet. The web development process includes web design, web content development, client-side/server-side scripting and network security configuration, among other tasks. Web development is also known as website development and Web development is the coding or programming that enables website functionality, per the owner's requirements.
